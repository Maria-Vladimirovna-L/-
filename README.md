# - GitHub User Finder

## Автор
Баринов Никита Андреевич

## Описание
Приложение с графическим интерфейсом для поиска пользователей GitHub по имени. Позволяет добавлять пользователей в избранное и сохранять их в локальный файл JSON. Использует GitHub API.

## Требования
- Python 3.8+
- Библиотеки: requests, tkinter, json

## Установка
1. Клонируйте репозиторий:
   git clone https://github.com/yourname/github-user-finder.git
2. Перейдите в папку проекта.
3. Установите зависимости:
   pip install -r requirements.txt

## Использование
1. Запустите приложение:
   python main.py
2. Введите имя пользователя в поле поиска и нажмите "Поиск".
3. Выберите пользователя из списка и нажмите "Добавить в избранное".
4. Избранные пользователи сохраняются в файл favorites.json.

## Как использовать API
Приложение использует публичный GitHub API: https://api.github.com/users/{username}
Ограничения: 60 запросов в час для неавторизованных пользователей.

## Примеры использования
- Поиск пользователя: `octocat`
- Добавление в избранное: выберите пользователя из списка и нажмите кнопку.
